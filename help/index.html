Launcher

How the Main Launcher Works

The main Video Editing Tools by Editopia window is the home screen for the bundle. Each card launches a tool or management dialog, while the launcher keeps shared controls such as Help, Preferences, version display, updates, and Premiere auto-minimize.

Top-right controls

Preferences opens the settings editor, the version label shows the current build, and Help opens the published help route.

Tool cards

Each card shows the tool title, a concise purpose line, an availability state, and its launch or management button.

Status footer

The footer reports tool launches, saves, update messages, and short operational feedback.

Window handoff

The launcher can hide, minimize, or stay visible when a tool opens, depending on your preference.

Preferences link Help link Version label Launch status Availability checks
If a card says Payload missing or Script missing, the build on that machine is incomplete or outdated. Reinstalling with the current setup is usually the cleanest fix.
Quick Start

Which Tool Should You Open?

Folder Restorer

Use it when Windows folders show the wrong icon and you want default folder icons back.

Video Transcoder

Use it when you want to scan a folder for Premiere-trouble media and optionally re-encode it.

Folder Sync

Use it when you need to copy only new or changed media from one folder tree into another.

Auto Sequencer

Use it when you need to install or manage the bundled Premiere CEP extension.

Tool Guide

Folder Restorer

Folder Restorer repairs default Windows folder icons. It removes icon-related desktop.ini overrides, clears related attributes, refreshes icon state, and restarts Windows Explorer so the visual change actually appears.

Main features

  • Accepts a pasted path or a built-in folder chooser.
  • Recursively scans subfolders under the selected root.
  • Shows counts for folders scanned, matches found, removals, and warnings.
  • Supports dry run and optional elevation for live changes.

How to use it

  1. Open Folder Restorer from the launcher.
  2. Paste the path or click Choose Folder.
  3. Click Restore Icons.
  4. Confirm the Explorer restart warning.
  5. Review the summary after the run finishes.
Open File Explorer windows will close during the Explorer restart. If some folders still look wrong after the run, a manual Explorer restart or full sign-out can help refresh the icon cache.
Tool Guide

Video Transcoder

Video Transcoder watches a folder, inspects media with ffprobe, flags formats that match your unsupported lists, and can optionally create safer outputs with ffmpeg. It is a compatibility checker first and an automation helper second.

Main features

  • Choose a watch directory and scan it once or continuously.
  • Toggle recursive scanning for subfolders.
  • Control the polling interval.
  • Edit watched extensions, unsupported video codecs, unsupported audio codecs, and unsupported containers.
  • Enable automatic encoding and optionally replace the original.
  • Writes rolling logs to Logs.

How to use it

  1. Open Video Transcoder.
  2. Set the watch folder.
  3. Confirm the ffprobe and ffmpeg paths.
  4. Decide whether you want manual scans or continuous monitoring.
  5. Enable Encode when flagged only if you want automatic output creation.
  6. Click Scan Now or Start.
  7. Use Save Settings to keep the current setup.

FFmpeg setup help

  • Runs a startup dependency check each time.
  • If tools are missing, it can prompt for a guided FFmpeg download.
  • Installs binaries into tools\ffmpeg\bin and saves the paths into config.

Useful behavior

  • Flagged videos can be re-encoded to H.264 with AAC audio and +faststart.
  • Flagged images can be converted to PNG.
  • Use the Hide button or press Ctrl+Shift+H to hide or restore the window.

Folder Sync

Folder Sync copies only new or changed media files from a source folder into a target folder. It preserves the source structure, skips files that already appear current, and leaves unrelated target files alone instead of trying to force an exact mirror.

Main features

  • Source and target path fields with browse buttons.
  • Dry run only for a safe preview.
  • File-size and modified-time checks to avoid unnecessary copies.
  • Automatic creation of missing target subfolders.

How to use it

  1. Open Folder Sync.
  2. Select the source media folder.
  3. Select the target folder.
  4. Turn on Dry run only if you want a preview first.
  5. Click Sync Media Files.
  6. Review the summary for copied, updated, skipped, created, and error counts.
Folder Sync is one-way and non-destructive toward unrelated target files. It is for practical media syncing, not for exact bidirectional mirroring.
Tool Guide

Auto Sequencer Manager

Auto Sequencer is managed directly inside the launcher. It opens a dedicated dialog for the bundled Premiere CEP extension so you can install or repair it, save settings, open support folders, review event logs, and launch Premiere.

Main features

  • Install or repair the extension in the current user CEP folder.
  • Enable Adobe debug mode automatically.
  • Detect and launch Premiere.
  • Save masterFolder, poll interval, stable file age, and auto-start behavior.
  • Open the CEP folder and events folder directly.
  • Review recent event summaries and latest event JSON details.

How to use it

  1. Click Manage Auto Sequencer in the launcher.
  2. Review the extension status and Premiere path.
  3. Click Install / Repair or Install + Launch.
  4. Adjust the folder and timing settings if needed.
  5. Save the settings.
  6. In Premiere, open Window > Extensions (Legacy) > Auto Sequencer.
Auto Sequencer management and Premiere auto-minimize are separate features. One manages a Premiere extension. The other only controls whether Premiere is minimized when it becomes active.
Automation

Built-In Helpers Around the Tools

Premiere Auto-Minimize

  • Starts automatically with the launcher.
  • Shows live status text in the launcher.
  • Can be paused or resumed from the launcher button.
  • Supports the hotkey Ctrl+Alt+P.

Launcher return behavior

  • If the launcher hides or minimizes while a tool is open, it can restore itself after that tool closes.
  • This keeps the launcher useful as a hub instead of just an opener.
Preferences

Preferences and Update Behavior

The launcher includes a preferences window for update policy and how the launcher behaves when a tool opens.

Update controls

  • Enable or disable automatic updates.
  • Choose whether the app checks on launch.
  • Choose whether updates auto-download.
  • Choose whether downloaded installers auto-install.
  • Select stable, testing, or development.
  • Override the manifest URL if needed.

Tool launch window modes

  • hide keeps the launcher out of sight.
  • minimize keeps it on the taskbar.
  • stay_visible leaves it open during tool use.
The installer normally targets a per-user location under {localappdata}\Video Editing Tools, which avoids unnecessary machine-wide install friction on most Windows setups.
In Progress

Soundbite Locator Status

Soundbite Locator already has a local planning UI, but it is still considered in-progress tooling rather than a released launcher feature. The launcher keeps its card disabled on purpose until the workflow is fully checked and approved.

What already exists

  • Add local media files.
  • Enter one keyword or phrase per line.
  • Choose transcript mode, match mode, Premiere action, and context seconds.
  • Preview and save the search plan into config.

What is still coming

  • Transcript generation.
  • Timestamp hit extraction.
  • Marker-ready Premiere output.
  • Later clip-placement support.
Treat Soundbite Locator as a development feature for now. It should not be presented like a normal released launcher tool until its full flow is validated.

Troubleshooting

A tool card is disabled unexpectedly

If it is not Soundbite Locator, reinstall the latest build. Missing payloads usually mean the app bundle is incomplete.

Video Transcoder cannot start

Check the dependency prompt, confirm the watch folder exists, and make sure ffmpeg.exe and ffprobe.exe are available.

Folder Restorer needs permission

Live changes may require admin rights depending on the target folder. Use the elevation path for protected locations.

Auto Sequencer does not show in Premiere

Run Install / Repair, verify Premiere was detected, then open Window > Extensions (Legacy) > Auto Sequencer.

Folder Sync copied less than expected

It only targets supported media files and skips files that appear unchanged. Run a dry run first if you want to confirm the selection logic.
