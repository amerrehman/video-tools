Launcher Overview

The main Video Editing Tools by Editopia window acts like a dashboard. Each button starts the original script for that tool in its own working folder, so the launcher itself stays simple while the individual tools keep their own logic.

One Window

Open the utility you need without browsing around different folders or scripts.

Windows Focused

The launcher and bundled helpers are built around the Windows environment and Explorer workflow.

Separate Scripts

Each button launches the original script or packaged tool instead of duplicating the tool logic inside the launcher.

Included Tools

Folder Restorer

Repairs normal folder icons by removing custom icon overrides and refreshing Windows Explorer.

Video Transcoder

Launches the external encoder workflow so video files can be converted into a format that behaves better across apps and devices.

Folder Sync

Copies only new or changed files from a source folder into a target folder while leaving unrelated target files alone.

Premiere Auto-Minimize

The launcher starts a Premiere auto-minimize controller when the window opens. Its status text updates live, and the toggle button lets you pause or resume that behavior without leaving the launcher.

  • Use the built-in toggle button if you want the helper temporarily paused.
  • The status message in the launcher tells you whether it is active and which hotkey label is currently configured.
  • If Premiere-specific automation is not needed, you can leave the launcher open and ignore that panel.

Install Notes

The published installer is VideoEdittingTools.exe. It installs the launcher and the packaged utilities into a per-user location under {localappdata} by default, which avoids the extra friction of requiring machine-wide admin access for normal installs.

  • Download the installer from the release page at /video-tools/.
  • Run the setup and keep the default location unless you have a reason to change it.
  • After install, open Video Editing Tools by Editopia and launch the utility you need from the dashboard.

Troubleshooting

  • If a launch button reports that a script is missing, reinstall with the latest setup.exe.
  • If Folder Sync or Folder Restorer fails unexpectedly, try closing Windows Explorer-heavy workflows and reopening the app.
  • If the Video Transcoder button does nothing, the external transcoder script path may not be available on that machine.
  • If you are reviewing a release online, return to the downloads page to compare the current installer and release notes.
