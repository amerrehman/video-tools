Tools

Included Tools and What They Do

Folder Restorer

Repairs normal Windows folder icons by recursively removing icon-related desktop.ini overrides, clearing attributes, refreshing icon state, and restarting Explorer.

  • Accepts a pasted path or a built-in folder picker.
  • Skips hidden and system-heavy folders like $Recycle.Bin.
  • Shows a summary of folders scanned, removals, and warnings.
  • Supports dry run and optional elevation for live changes.

Video Transcoder

Watches a folder, checks media with ffprobe, flags formats that are likely to cause trouble in Premiere, and can create safer outputs with ffmpeg.

  • Scans video files and common image formats.
  • Lets you define blocked codecs, containers, scan extensions, and output extension.
  • Supports manual scans, continuous monitoring, recursion, and optional replacement of the original.
  • Creates rolling logs and supports Ctrl+Shift+H to hide or restore the app.

Folder Sync

Copies new or changed media files from a source folder into a target folder while leaving unrelated target files alone.

  • Preserves the source folder structure under the target.
  • Uses file size and modified time to skip unchanged items.
  • Has a dry-run mode to preview the result first.
  • Blocks invalid path combinations, including nested source/target pairs.

Auto Sequencer Manager

Managed directly from the launcher, Auto Sequencer installs the bundled Premiere CEP extension, saves its config, opens support folders, reads recent events, and can launch Premiere.

  • Installs or repairs the extension in the current user CEP folder.
  • Enables Adobe debug mode automatically.
  • Edits the master folder, poll interval, stable file age, and auto-start behavior.
  • Shows recent event summaries plus raw JSON details for the latest event.
Automation

Built-In Automation and Convenience Features

Premiere Auto-Minimize

The launcher starts a background controller that watches the foreground window and minimizes Adobe Premiere Pro when it becomes active.

  • Starts automatically with the launcher.
  • Can be paused or resumed from the launcher button.
  • Also supports the hotkey Ctrl+Alt+P.
  • Shows live status text in the launcher.

FFmpeg Runtime Setup

If the Video Transcoder cannot find ffmpeg.exe or ffprobe.exe, it checks writable locations, explains what is missing, and can prompt for a guided FFmpeg download into tools\ffmpeg\bin.

  • Runs startup dependency checks.
  • Shows download progress in a dedicated window.
  • Writes installed paths back into the transcoder config.

Auto Sequencer install + launch flow

From one dialog you can install or repair the extension, open the CEP folder, open the events folder, detect Premiere, and run an Install + Launch action.

Safer media outputs

When transcoding is enabled, videos are re-encoded to H.264 with AAC audio and +faststart, while images are converted to PNG. Existing up-to-date outputs are reused.

Auto Sequencer and Premiere auto-minimize are separate features. One manages a Premiere extension and its watch settings; the other only controls whether the Premiere window is minimized when it becomes active.
Preferences

Preferences, Updates, and Launcher Behavior

The app now has a dedicated Preferences window for update behavior and launcher handoff behavior.

Update settings

  • Enable or disable automatic updates.
  • Choose whether the app checks on launch.
  • Choose whether updates auto-download.
  • Choose whether downloads auto-install.
  • Select stable, testing, or development.
  • Override the manifest URL if needed.

Launcher window options

  • hide keeps the launcher out of sight while a packaged tool is open.
  • minimize keeps it on the taskbar but out of the way.
  • stay_visible leaves the launcher open during tool use.

Update flow

On launch, the app can check the configured manifest, compare versions, download the new installer into an updates folder, and optionally schedule installation before restarting.

Workflow

A Typical Way to Use the App

1. Open the launcher

The app shows the current version, starts Premiere auto-minimize, and can check for updates based on your preferences.

2. Choose the tool you need

Use Folder Restorer for broken folder icons, Folder Sync for media mirroring, Video Transcoder for compatibility scanning and re-encoding, or Auto Sequencer for Premiere extension management.

3. Save settings where it matters

The transcoder keeps its own monitoring config, Auto Sequencer keeps its watch settings, and the launcher keeps shared app preferences.

4. Review logs and status

The launcher footer, transcoder logs, Auto Sequencer event view, and end-of-task summaries help you confirm what happened.

The installer normally targets a per-user install under {localappdata}\Video Editing Tools, which keeps setup simple on most Windows systems.
Troubleshooting

Common Issues and What to Check

A launch button says something is missing

Reinstall with the latest setup. Missing bundled payloads usually point to an incomplete install or an outdated build.

Video Transcoder will not start scanning

Check the watch folder path and confirm FFmpeg tools are available. The app can prompt for an FFmpeg download if needed.

Auto Sequencer does not appear in Premiere

Run Install / Repair from the manager, then open Premiere and go to Window > Extensions (Legacy) > Auto Sequencer.

Folder Restorer needs extra permission

Use its elevation flow when live changes require administrator access. The tool can relaunch itself with admin rights.

This app is built around Windows behavior, Windows Explorer, Windows hotkeys, and Adobe Premiere on Windows. Some features will not apply on other operating systems.
